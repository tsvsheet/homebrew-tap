# homebrew-tap

The [Homebrew](https://brew.sh) tap for the [tsvsheet](https://tsvsheet.com) CLIs (macOS and Linux):

```sh
brew tap tsvsheet/tap
brew install tsvsheet   # a spreadsheet for plain text
brew install isnow      # match instants against isnow date/time patterns
```

Formulas in [Formula/](Formula/) are generated by [scripts/update.sh](scripts/update.sh) from the latest [tsvsheet.go](https://github.com/tsvsheet/tsvsheet.go) and [isnow.go](https://github.com/tsvsheet/isnow.go) releases — binaries built and attested by GitHub Actions in those repos, pinned here by sha256. A scheduled workflow keeps them current; never edit a formula by hand. See [tsvsheet.com/install](https://tsvsheet.com/install/) for every install method.
